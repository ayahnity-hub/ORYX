# ORYX

## Project Overview

ORYX is an AI-powered real-time security and threat detection system developed using Artificial Intelligence and Computer Vision technologies.

The system provides:

- Real-time face recognition
- Live surveillance monitoring
- Threat and suspect detection
- License plate recognition
- GPS tracking and patrol monitoring
- OTP authentication system
- Role-based access control for Admin, Police, and Officers

The project integrates Flask, MongoDB, InsightFace, YOLO, and OCR technologies to build a complete smart security platform.

---

## Dataset Description

The system uses the SCface dataset for face recognition and surveillance testing.

Dataset Link:

https://www.kaggle.com/datasets/yazkarajih/scface

The dataset contains:

- Mugshot frontal images
- Rotated face images
- Surveillance camera images
- Multiple distance variations

## Installation Steps

Clone the repository:

```bash
git clone https://github.com/your-username/ORYX.git
```

Move into the project directory:

```bash
cd ORYX
```

Install required libraries:

```bash
pip install -r requirements.txt
```

## Environment Setup

Required Technologies:

- Python 3.11
- Flask
- MongoDB
- OpenCV
- InsightFace
- YOLO
- EasyOCR
- PaddleOCR

## How to Run

Run the Flask application:

```bash
python app.py
```

Then open:

```text
http://127.0.0.1:5000
```
